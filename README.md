# Study Notion (MERN)

A full-stack MERN application for managing study content, courses, and user progress.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User authentication
- Course management
- Progress tracking
- Responsive UI

## Installation

```bash
git clone https://github.com/RuchiYadav2006/study_notion.git
cd study_notion
npm install
